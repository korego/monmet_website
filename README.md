# Monmet Technologies - Modern Website

A modern, responsive website for Monmet Technologies - manufacturer of high-performance fireplace fans and blowers.

## Features

- **Modern Design**: Clean, contemporary interface with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Mobile hamburger menu
  - Animated cards and sections
  - Contact form with validation
  - Back-to-top button
  - Parallax effects
- **Product Showcase**: Organized product listings by manufacturer
- **Performance Optimized**: Fast loading and smooth interactions

## Pages

- **Home** (`index.html`): Landing page with hero section, features, about, products overview, news, and contact form
- **Products** (`products.html`): Complete product catalog organized by manufacturer with filtering
- **About** (`about.html`): Company information, mission, values, certifications, and location

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons

2. Open `index.html` in a web browser

## Customization

### Colors

Edit CSS variables in `css/style.css`:

```css
:root {
  --primary-color: #ff6b35;
  --secondary-color: #004e89;
  --dark-color: #1a1a2e;
  /* ... */
}
```

### Content

- Update text content directly in HTML files
- Modify contact information in footer and contact sections
- Add/remove products in `products.html`

### Images

Replace placeholder icons with actual images by updating the image sections in HTML files.
